## 帝国行政体系（近代体制版）

### 一级：中央与高等区划

> 官方直属区与最高地方权力单位
> 包含六类：**帝京、京畿、府、司、昏、国**

| 一级单位   | 职能 / 定位                | 典型人口规模     | 备注          |
| ------ | ---------------------- | ---------- | ----------- |
| **帝京** | 首都直辖区，中央政府与帝宫所在地       | 100万–200万  | 相当于“帝国首都特区” |
| **京畿** | 帝京周围的直属行政区，供应首都、驻防中枢   | 300万–1000万 | 类似近代“首都圈省”  |
| **府**  | 一般省级区划，直属中央管理          | 50万–500万   | 核心官治区       |
| **司**  | 专责管理区（边防、贸易、移民、资源、交通等） | 不定         | 功能化区域       |
| **昏**  | 国家级宗教与祭祀保留地            | 少于10万      | 禁民居、属宗庙管理   |
| **国**  | 封国 / 藩国 / 民族属国         | 50万–800万   | 皇族或少数民族领地   |

---

### 二级：直属区划（由中央设定）

> 一级单位的内部主要分区，由中央统一命名与立法范围。

| 一级所属 | 二级单位                | 职能            |
| ---- | ------------------- | ------------- |
| 帝京   | 帝宫、帝卫、京台            | 皇城、禁军、都政机构所在地 |
| 京畿   | 州                   | 地方行政区         |
| 府    | 城中（省府驻地）、府京（府直辖区）、州 | 行政中心 + 普通地方区  |
| 司    | 驻办（机构直辖区）、附司、州      | 边防/专责区划       |
| 昏    | 堂中、昏卫               | 祭祀与守护         |
| 国    | 国都、国京、国州、国昏         | 封国内部政区与宗庙区    |

---

### 三级：地方基层行政单位（官方定编）

> 属于中央或封国的“最低一级正式行政区”，但并非自治级别。

| 名称    | 职能 / 特征            | 常见人口规模         | 说明          |
| ----- | ------------------ | -------------- | ----------- |
| **县** | 一般行政与民政核心          | 5万–20万         | 基本民政区       |
| **卫** | 军政合署地区             | 1万–10万（多为常备兵区） | 防务重地        |
| **镇** | 商贸与交通节点            | 5千–5万          | 多位于驿道、水陆要冲  |
| **港** | 港口与对外贸易区           | 2千–10万         | 可隶属府、司、国等上级 |
| **堡** | 要塞或边防屯地            | 数百–数千（军户为主）    | 边地要地        |
| **台** | 行政/通讯/观测节点         | 1千–1万          | 有技术与官署功能    |
| **方** | 农业/手工业聚落区          | 5百–2万          | 民生产区        |
| **口** | 关隘、出入口、通道管理区       | 数百–数千          | 交通管制与通行稽查   |
| **角** | 偏远或独立自然区，常设监理或宗族自治 | 数百–数千          | 地理或族群特殊区    |
| **占** | 特许开发区或资源占领区        | 变动             | 经济/殖产性质     |

**在封国中：**

* 以上第三级单位的名称与权限可由封国内部自定；
* 封君可将州直接分为府、县、镇等，也可废除部分称谓或自创（例如“领”“部”“旗”）；
* 中央只要求其第三级单位具备明确户籍、税籍与军籍统计。

---

### 四级：基层自治层（非官方）

> 属于地方、民间或宗族自治组织。
> 不纳入官方行政分层，仅在统计和警务中出现。

常见形式：

* **村、社、坊、区、段、巷、团、营** 等。
* 人口一般在 **几十到数千**，
  由地方自治团体、宗族议会或行业公会管理。
* 不设常备官职，必要时由第三级（县/镇等）派员兼理。
